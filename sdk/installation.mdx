## Installation

Install the Limits SDK using your preferred package manager:

<CodeGroup>
```bash npm
npm install @limits/sdk
```

```bash yarn
yarn add @limits/sdk
```

```bash pnpm
pnpm add @limits/sdk
```
</CodeGroup>

<Warning>
  The SDK requires Node.js 18+ for native `fetch` support.
</Warning>

## Configuration

Initialize the client with your API key:

```typescript
import { Limits } from "@limits/sdk";

const limits = new Limits({
  apiKey: process.env.LIMITS_API_KEY,
});
```

### Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `apiKey` | `string` | Required | Your Limits API key (must start with `sk_`) |
| `debug` | `boolean` | `false` | Enable debug logging for troubleshooting |



## Get Your API Key

1. Sign up at [app.limits.dev](https://app.limits.dev)
2. Click on the **User Menu** (your profile section with the person icon at the bottom left of the dashboard)
3. In the settings modal that opens, click on **API Keys** (with the `<>` icon) in the left sidebar
4. Click **+ Generate API Key** to create a new API key
5. Add it to your environment variables

```bash
export LIMITS_API_KEY="sk_your_api_key_here"
```

<Warning>
  Never commit your API key to version control. Use environment variables or a secrets manager.
</Warning>

## Quick Example

```typescript
import { Limits } from "@limits/sdk";

const limits = new Limits({
  apiKey: process.env.LIMITS_API_KEY,
  debug: true, // Optional: enable debug logging
});

// Check conditions with structured data
const decision = await limits.checkConditions('amount-minimum-usd', {
  amount: 500,
  currency: 'USD',
});

if (decision.isBlocked) {
  console.error('Policy blocked:', decision.data.reason);
} else {
  console.log('Policy allowed:', decision.data.reason);
}
```
