The Limits API allows you to programmatically manage policies, agents, and check policy decisions.

## Base URL

```
https://api.limits.dev/v1
```

## Authentication

All API requests require an API key passed in the `Authorization` header:

```bash
curl https://api.limits.dev/v1/agents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Rate Limits

| Plan | Requests/month |
|------|---------------|
| Free | 1,000 |
| Startup | 100,000 |
| Enterprise | Unlimited |

## Endpoints

### Policy Check

```bash
POST /v1/check
```

Check if an action is allowed by policy:

```bash
curl -X POST https://api.limits.dev/v1/check \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent": "support-bot",
    "action": "send_email",
    "context": {
      "recipient": "user@example.com"
    }
  }'
```

### Response

```json
{
  "allowed": true,
  "policyId": "pol_abc123",
  "policyVersion": "1.0.0",
  "timestamp": "2025-01-18T12:00:00Z"
}
```
