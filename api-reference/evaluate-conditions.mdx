---
title: "Evaluate conditions"
description: "Evaluate a conditions policy with structured input"
openapi: "api-reference/openapi.yaml POST /api/policies/{policyKey}/evaluate/conditions"
---

Evaluates a **conditions** policy by key. Send a JSON body with `request.input` as an object (e.g. amount, currency, user fields). The API returns the evaluation result with `action` (ALLOW, BLOCK, or ESCALATE), `success`, and `message`.
