---
title: "Evaluate policy"
description: "Evaluate a policy by key with the given input (request/response/text)"
openapi: "api-reference/openapi.yaml POST /api/policies/{policyKey}/evaluate"
---

Evaluates a policy by key with the given input. The body can include `request` (structured data for condition evaluation), `response`, and/or `text` (for LLM-based policies). Returns the evaluation result with action (ALLOW, BLOCK, or ESCALATE).
