---
title: "Evaluate guardrails"
description: "Evaluate a guardrails policy on text"
openapi: "api-reference/openapi.yaml POST /api/policies/{policyKey}/evaluate/guardrails"
---

Evaluates a **guardrails** policy by key. Send a JSON body with `request.input` as a string (the text to check, e.g. model output or user message). The API returns the evaluation result with `action` (ALLOW, BLOCK, or ESCALATE), `success`, and `message`.
