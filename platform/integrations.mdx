Connect Limits with third-party services to extend functionality and receive notifications.

## Available Integrations

### Supported now

- **Webhooks** — Send policy notifications (Allow, Block, Escalate) to any HTTP endpoint. Configure webhook URLs in your policies. See [Notifications](/platform/notifications) for setup and payload details.

### Coming soon

- **Slack** — Send notifications to Slack channels. Coming in a future release.

---

## Slack — Coming Soon

<Info>
**Upcoming:** Slack integration is in development. When it ships, you will be able to connect your Slack workspace and send policy notifications to channels. Below is a preview.
</Info>

### Planned Setup

1. Navigate to the Integrations page
2. Find the Slack integration card
3. Click the toggle or "Connect" button
4. Authorize with your Slack workspace
5. Select channels for notifications
6. Configure channels per action (Allow, Block, Escalate) in policy Notifications

### Planned Notification Content

Slack notifications will include:

- Policy name and key
- Action taken (Allow / Block / Escalate)
- Request details
- Evaluation reason
- Timestamp

We’ll update this page and the Notifications docs when Slack is available.
