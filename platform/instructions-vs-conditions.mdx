---
title: "Instructions vs Conditions"
description: "Two ways to define a policy: structured conditions or natural-language instructions"
---

When you **define a policy** in the platform, you choose **one** of two modes:

|             | **Conditions mode**                            | **Instructions mode**                      |
| ----------- | ---------------------------------------------- | ------------------------------------------ |
| **Answers** | _When_ or _if_ something applies               | _What_ to do (required action or behavior) |
| **Format**  | Structured rules (e.g. field, operator, value) | Natural language (text, bullets, steps)    |
| **SDK**     | `limits.check()`                               | `limits.evaluate()`                        |

## Conditions mode

Define **when** a request is allowed, blocked, or escalated using structured rules:

- Block **if** `request.amount > 5000`
- Escalate **when** `user.risk_level == 'high'`
- Allow **when** neither of the above matches

Use `limits.check()` with structured data:

<CodeGroup>
```javascript JavaScript
const decision = await limits.check('your-policy-key', {
  amount: 100,
  user: { risk_level: 'low' },
});

if (decision.isAllowed) console.log('Allowed');
if (decision.isBlocked) console.log('Blocked');
if (decision.isEscalated) console.log('Escalated');

````
```typescript TypeScript
const decision = await limits.check('your-policy-key', {
  amount: 100,
  user: { risk_level: 'low' },
});

if (decision.isAllowed) console.log('Allowed');
if (decision.isBlocked) console.log('Blocked');
if (decision.isEscalated) console.log('Escalated');
````

</CodeGroup>

## Instructions mode

Describe **what** should happen in natural language:

- "Block requests over $10,000."
- "Escalate when the user is high risk."
- "Allow only if the resource is in the approved list."

Use `limits.evaluate()` with the user's prompt and your AI agent's response:

<CodeGroup>
```javascript JavaScript
// User asks your AI agent for a list of countries
const prompt = 'List all countries we can ship to';
const agentResponse = { text: 'We can ship to: USA, Canada, Uganda, UK, France' };

// Evaluate the agent's response against your policy
// (e.g., policy instructions: "Block if country X is mentioned")
const decision = await limits.evaluate(
'your-policy-key',
prompt,
agentResponse
);

if (decision.isAllowed) console.log('Allowed');
if (decision.isBlocked) console.log('Blocked');
if (decision.isEscalated) console.log('Escalated');

````
```typescript TypeScript
// User asks your AI agent for a list of countries
const prompt = 'List all countries we can ship to';
const agentResponse = { text: 'We can ship to: USA, Canada, Uganda, UK, France' };

// Evaluate the agent's response against your policy
// (e.g., policy instructions: "Block if country X is mentioned")
const decision = await limits.evaluate(
  'your-policy-key',
  prompt,
  agentResponse
);

if (decision.isAllowed) console.log('Allowed');
if (decision.isBlocked) console.log('Blocked');
if (decision.isEscalated) console.log('Escalated');
````

</CodeGroup>
