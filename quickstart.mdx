## Install the SDK

<CodeGroup>
```bash npm
npm install @limits.dev/sdk
```

```bash yarn
yarn add @limits.dev/sdk
```

```bash pnpm
pnpm add @limits.dev/sdk
```
</CodeGroup>

## Initialize the Client

```typescript
import { Limits } from "@limits.dev/sdk";

const limits = new Limits({
  apiKey: process.env.LIMITS_API_KEY,
});
```

## Check a Policy

Before your agent performs an action, check if it's allowed:

```typescript
const result = await limits.check({
  agent: "support-bot",
  action: "send_email",
  context: {
    recipient: "user@example.com",
    subject: "Support ticket update",
  },
});

if (result.allowed) {
  // Proceed with the action
  await sendEmail(recipient, subject, body);
} else {
  console.log("Action blocked:", result.reason);
}
```
