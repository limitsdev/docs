## Install the SDK

<CodeGroup>
```bash npm
npm install @limits/sdk
```

```bash yarn
yarn add @limits/sdk
```

```bash pnpm
pnpm add @limits/sdk
```
</CodeGroup>

## Initialize the Client

```typescript
import { Limits } from "@limits/sdk";

const limits = new Limits({
  apiKey: process.env.LIMITS_API_KEY,
});
```

## Check Conditions

Check policies with structured data:

```typescript
const decision = await limits.checkConditions('amount-minimum-usd', {
  amount: 500,
  currency: 'USD',
  customerId: 'cus_123',
});

if (decision.isBlocked) {
  console.error('Action blocked:', decision.data.reason);
  return;
}

// Proceed with the action
console.log('Action allowed:', decision.data.reason);
```

## Check Interactions

Check policies with text-based interactions (e.g., LLM responses):

```typescript
const decision = await limits.checkInteractions(
  'block-amount-under-9000',
  "Acknowledged. I tried to submit your payment for Order ORD-938271..."
);

if (decision.isBlocked) {
  console.error('Interaction blocked:', decision.data.reason);
} else {
  // Proceed with the interaction
}
```

## Complete Example

```typescript
import { Limits } from "@limits/sdk";

const limits = new Limits({
  apiKey: process.env.LIMITS_API_KEY,
  debug: true, // Optional: enable debug logging
});

// Check conditions
const decision = await limits.checkConditions('amount-minimum-usd', {
  amount: 500,
  currency: 'USD',
});

// Use boolean helpers for cleaner conditionals
if (decision.isBlocked) {
  console.error('Blocked:', decision.data.reason);
} else if (decision.isEscalated) {
  console.warn('Escalated:', decision.data.reason);
} else if (decision.isAllowed) {
  console.log('Allowed:', decision.data.reason);
}

// Check for validation errors
if (decision.errors && decision.errors.length > 0) {
  decision.errors.forEach(error => console.error(`  - ${error}`));
}
```
